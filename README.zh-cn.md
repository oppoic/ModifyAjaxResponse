## 本地安装
1. [下载](https://github.com/oppoic/ModifyAjaxResponse/releases) 最新的 **ModifyAjaxResponse.zip**
2. 打开Chrome浏览器，地址栏访问：chrome://extensions/
3. 打开"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择源码目录：ModifyAjaxResponse/src
6. 确认

## 简介
仅重写xhr和fetch方法的返回实体，F12开发者工具-网络面板里的返回值不会变
